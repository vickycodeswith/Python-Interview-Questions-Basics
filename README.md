A beginner-friendly collection of **Python interview questions and solutions** created for practicing core Python concepts.
This repository is useful for students, freshers, and anyone preparing for Python coding rounds or technical interviews.

---

## 📌 About This Repository

This repository contains basic Python practice questions solved using **Jupyter Notebook**.
The questions are focused on building strong fundamentals in Python programming.

Topics include:

* Lists
* Strings
* Functions
* Dictionaries
* Object-Oriented Programming (OOPs)
* Basic coding logic for interviews

---

## 🚀 Topics Covered

| Topic        | What You Will Practice                                         |
| ------------ | -------------------------------------------------------------- |
| Lists        | Searching, sorting, duplicates, max/min, reverse, count, merge |
| Strings      | Reverse string, vowels, palindrome, character counting         |
| Functions    | Function creation, parameters, return values, reusable logic   |
| Dictionaries | Key-value pairs, accessing data, updating data, loops          |
| OOPs         | Classes, objects, inheritance, methods, encapsulation basics   |

## 🧠 Example Questions

Some examples of questions included in this repository:

* Find the maximum number in a list without using `max()`
* Reverse a string
* Count vowels in a string
* Find even numbers from a list
* Remove duplicate elements from a list
* Find the second largest number
* Count even and odd numbers
* Move zeroes to the end of a list
* Find common elements in two lists
* Create basic classes and objects using OOPs
